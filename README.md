# System-Programming

## Assembler
SIC/XE source를 Object Program Code로 바꾸는 어셈블러를 만들기 위한 프로젝트.

다양한 프로그래밍 언어(C언어 버전, JAVA버전, Python버전)를 사용하여 SIC/XE source를 line별로 처리하여 Object Code로 바꿔 보았습니다.
<br>
## Simulator
자바 GUI를  object코드를 분석 후 명령어, Target Address, Register의 값 등을 순차적으로 확인 할 수 있는 시뮬레이터를 구현하였습니다.
